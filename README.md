# 🌍 Tourism Review Sentiment Classification using Machine Learning

## 📌 Project Overview

This project is a Machine Learning and Natural Language Processing (NLP) application that classifies tourism reviews into Positive, Negative, or Neutral sentiments.

A Flask web application was developed and deployed on PythonAnywhere to provide real-time sentiment prediction.

---

## 🚀 Features

- Text Cleaning and Preprocessing
- TF-IDF Feature Extraction
- Multiple Machine Learning Models
- Sentiment Prediction
- Flask Web Application
- Live Deployment

---

## 🤖 Machine Learning Models

- Logistic Regression
- Multinomial Naive Bayes
- Decision Tree
- Random Forest

---

## 📊 Evaluation Metrics

- Confusion Matrix
