About This Project

This Tourism Review Sentiment Analysis project predicts whether a tourism review is Positive or Negative using Machine Learning. The application uses TF-IDF Vectorization and Logistic Regression for text classification. Flask provides the backend while HTML, CSS and JavaScript create a modern responsive user interface.

How It Works

Simple Machine Learning Workflow

⚙️

📊

Result

Machine Learning Technologies

Technologies used in this project

📄 TF-IDF

Converts text into numerical vectors by measuring the importance of each word.

📈 Logistic Regression

A supervised learning algorithm used for binary sentiment classification.

🐍 Flask

Provides the backend and connects the trained model with the website.
